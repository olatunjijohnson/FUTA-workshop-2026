We’ll also define a small helper to create a mesh from point data.

Code 
make_spde_mesh <- function(df, max_edge = c(200000, 600000), cutoff = 50000) {
pts <- df |>
select(utm_x, utm_y) |>
unique()
coordinates(pts) <- ~ utm_x + utm_y
inla.mesh.2d(
loc = coordinates(pts),
max.edge = max_edge,
cutoff = cutoff
)
}
